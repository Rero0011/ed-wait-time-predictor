# ED Wait Time Predictor

## Overview
This project predicts emergenccy department wait times using machine learning (XGBoost)
and provides explainable insights using SHAP.

## Features
- Wait time prediction
- Explainable AI (SHAP)
- Interactive dashboard (Streamlit)

## How to Run
- pip install -r requirements.txt
- streamlit run app.py

## Dataset
Synthetic dataset simulating ED conditions.
